# Stock Price App
A simple web app that returns a company's stock price from a CSV dataset.

---

## What It Does
- Accepts a company name as input and returns the matching stock price
- Reads and parses stock data from a local CSV file
- Displays results in a lightweight client-side interface

## How It's Built
Node.js server loads the CSV file and handles incoming requests. Client-side JavaScript takes user input, queries the server, and updates the UI with the result. Built for a CS20 Web Programming course at Tufts.
